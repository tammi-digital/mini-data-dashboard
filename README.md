# Mini Data Dashboard

This Python project generates a mini sales dashboard using CSV data.

The script calculates:

- Total sales
- Average sale
- Highest sale
- Lowest sale
- Day of highest and lowest sale

Example output:

=== Mini Sales Dashboard ===
Total Sales: 1640
Average Sale: 234.29
Highest Sale: 420 on Thursday
Lowest Sale: 90 on Wednesday

Skills demonstrated:

- Python
- CSV file handling
- Data aggregation
- Summary statistics
- Simple reporting
